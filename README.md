# 📄 Resume Analyzer

An AI-powered full-stack web application that analyzes resumes and provides actionable feedback to help job seekers improve their chances of landing interviews.

---

## ✨ Features

- **Resume Parsing** — Upload your resume and extract key information automatically
- **AI-Powered Analysis** — Get intelligent feedback on content, structure, and clarity
- **Skill Gap Detection** — Identify missing or underrepresented skills for your target role
- **ATS Compatibility Check** — Ensure your resume passes Applicant Tracking Systems
- **Score & Suggestions** — Receive an overall resume score with specific improvement tips
- **Clean, Responsive UI** — Intuitive interface built with modern HTML, SCSS, and JavaScript

---

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm or yarn

---

### 1. Clone the Repository

```bash
git clone https://github.com/Platypus-coder07/resume---analyzer.git
cd resume---analyzer
```

---

### 2. Set Up the Backend

```bash
cd Backend
npm install
```

Create a `.env` file in the `Backend` directory and add any required environment variables (e.g., API keys):

```env
PORT=5000
# Add other variables as needed
```

Start the backend server:

```bash
npm start
```

---

### 3. Set Up the Frontend

```bash
cd ../Frontend
npm install
npm start
```

The app should now be running at `http://localhost:3000` (or whichever port is configured).

---

## 📸 Usage

1. Open the app in your browser
2. Upload your resume (PDF or DOCX)
3. Wait for the analysis to complete
4. Review your score, feedback, and suggestions
5. Make improvements and re-upload to track progress
